# Changelog

本文件记录 fate-frontend 前端项目的重要更新。

## [Unreleased] - 2026-02-04

### 优化 - iOS 风格日期时间选择器

统一首页和 Panel 页的日期时间选择器，使用 iOS 风格滚轮选择器替代原生 HTML5 控件，大幅提升移动端体验。

**新增文件：**
- `app/components/DatePicker.tsx` - iOS 风格滚轮日期选择器
- `app/components/form/dateTimeThemes.ts` - 主题配置文件

**修改文件：**
- `app/components/TimePicker.tsx` - 添加 theme 属性支持
- `app/components/Calender.tsx` - 使用新日期选择器替代原生控件
- `app/page.tsx` - 首页使用新的日期时间选择器

**功能特性：**
- 三列滚轮选择器（年/月/日），支持惯性滚动和自动吸附
- 年月日联动：自动调整日期（如 2 月天数变化）
- 中线选中框视觉反馈
- 清空和确认按钮
- 支持主题配置（landing 灰色调、panel 奶油色调）

**UI/UX：**
- 纯 Web 实现，所有平台表现一致
- 解决 Android 原生控件体验差的问题
- 触摸优化，移动端体验流畅
- 两套主题适配不同页面风格

## [2026-02-03] - 统计分析仪表盘

实现管理后台数据概览页面，展示用户、对话、消息等统计数据。

**新增文件：**
- `app/(auth)/admin/dashboard/page.tsx` - 数据概览页面

**修改文件：**
- `app/(auth)/admin/page.tsx` - 添加数据概览入口
- `package.json` - 添加 recharts 图表库依赖

**功能特性：**
- 统计卡片：总用户数、活跃用户、总对话数、待处理反馈
- 折线图：用户注册趋势（近30天）
- 饼图：用户来源分布（网页/小程序）
- 柱状图：对话量趋势（近7天）
- 实时刷新：支持手动刷新数据

**UI/UX：**
- 响应式布局，适配移动端
- 与管理后台风格一致
- 数据加载状态和错误提示

## [2026-02-02] - 找回密码功能

### 新增

实现用户找回密码功能的前端界面。

**新增文件：**
- `app/forgot-password/page.tsx` - 忘记密码页面
- `app/reset-password/page.tsx` - 重置密码页面

**功能特性：**
- 忘记密码表单：用户输入注册邮箱，发送重置链接
- 重置密码表单：用户通过邮件链接设置新密码
- 表单验证：邮箱格式验证、密码强度验证
- 用户反馈：发送成功/失败提示、重置成功跳转登录

**UI/UX：**
- 与登录页面风格一致
- 响应式设计，支持移动端
- 加载状态和错误提示
